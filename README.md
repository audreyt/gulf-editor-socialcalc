# gulf-socialcalc
This is a [gulf](https://github.com/marcelklehr/gulf) wrapper for the [socialcalc](https://npmjs.org/package/socialcalc) spreadsheet editor.
It utilizes the [ot-socialcalc](https://github.com/marcelklehr/ot-socialcalc) OT type.
